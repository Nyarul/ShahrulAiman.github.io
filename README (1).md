A personal CV portfolio website for Muhammad Shahrul Aiman Bin Mokhtar, a fresh graduate in Computer Science from Universiti Malaysia Terengganu.

- Dark & Modern design with neon green accents
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll animations

## How to Run Locally

1. Clone this repo:
   ```bash
   git clone https://github.com/yourname/portfolio.git
   ```
2. Open `index.html` in your browser — no build step needed!

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch → `/ (root)`
4. Your site will be live at `https://yourname.github.io/portfolio`

## Customise

Replace all `[ Placeholder ]` text in `index.html` with your real content:
- Projects (name + description)
- Certifications (name + issuer)
- Work experience details
- Contact links (email, GitHub, LinkedIn)
